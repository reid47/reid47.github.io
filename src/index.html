about me

I’m a software engineer and filmmaker currently living in San Francisco, CA. I’m passionate about creating things that make people’s lives a little easier or a little more enjoyable. My pronouns are he/him/his.

These days, I’m especially interested in front-end web development (JavaScript, React, etc), building accessible user interfaces, and programming language design (compiler-writing, type-checking, etc).

Prior to entering the “real world,” I grew up in a suburb of New York and went to school at Pomona College, where I majored in computer science and media studies.

recent projects

  • Pluma: I’m designing a statically-typed, functional programming language that compiles to JavaScript. It’s still very much a work-in-progress, but you can follow along with the project (and view the code) on GitHub.
  • Turnstyle: As an experiment to learn more about CSS-in-JS, I built my own CSS-in-JS library. This involved writing a CSS parser/preprocessor from scratch. It does automatic vendor-prefixing, has no dependencies, and supports server-side rendering. From my testing, it’s quite a bit smaller and faster than many of the other solutions out there. Check it out on GitHub.
  • Redraw: A pixel art tool built using React and the HTML5 Canvas. It’s meant to make creating pixel art easy and fun. Try it out, or browse the source code on GitHub.
